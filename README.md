네이버 영어사전 크롬 익스텐션
=============================

크롬 브라우저에서 영어 단어의 뜻과 예문을 쉽게 찾을 수 있도록 도와주는  
네이버 영어사전 크롬 익스텐션(Naver English Dictionary Chrome Extension)입니다.
